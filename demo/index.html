lSLider.js

Ultra simple jQuery slider Download

Installation

With Bower :

    
$ bower install lslider --save

How to use ?

In your HTML page, between <head> tags, import CSS from css/lslider.css

    
<link href="css/lslider.css" rel="stylesheet" type="text/css" media="screen" />

Next, import JavaScript files js/jquery.min.js and js/jquery.lslider.min.js :

    
<script type="text/javascript" src="js/jquery.min.js"></script>
<script type="text/javascript" src="js/jquery.lslider.min.js"></script>

Puts the following HTML code in your page :

    
<ul class="lslider">
    <li class="active"><img src="img/item-1.jpg" alt="Item 1" /></li>
    <li><img src="img/item-2.jpg" alt="Item 2" /></li>
    <li><img src="img/item-3.jpg" alt="Item 3" /></li>
    <li><img src="img/item-4.jpg" alt="Item 4" /></li>
</ul>

Finally, initialize the plugin at DOM ready :

    
$(document).ready(function() {
    $('.lslider').lslider({});
});

Options

Parameter Type Definition Values Default
animationSpeed int Animation speed to display image * 350
pause int Duration of one slide transition * 5000
beforeDisplay function Callback fired before display image function function(e, hiddenElement) { }
afterDisplay function Callback fired after display image function function(e, visibleElement) { }
